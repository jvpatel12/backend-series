forgotPasswordRequest

Accepts { email }.
Generates a temporary reset token, stores the hashed token and expiry on the user, sends a reset email (re-using your mail helper).
Returns 200 "Password reset email sent successfully".
resetPassword

Accepts { token, id, newPassword }.
Verifies token, expiry, updates the user's password, clears reset fields, and returns 200 "Password reset successfully".
changeCurrentPassword

Protected route (expects req.user from verifyJwt).
Accepts { currentPassword, newPassword }.
Verifies current password, updates to new password, returns 200 "Password changed successfully".
